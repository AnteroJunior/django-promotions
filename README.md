# Django Promotions

## Install

1. Start a virtual environment
`python3 -m venv venv`

2. Activate venv
`source venv/bin/activate`

3. Install Django
`python3 -m pip install Django`

4. Go to mypromotions folder
`cd mypromotions`

5. Run Django server
`python3 manage.py runserver`

Done! Enjoy the project.

## Technologies used
1. Django
2. Bootstrap 5
